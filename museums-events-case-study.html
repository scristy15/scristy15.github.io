UX / UI Case Study

Museums & Events App

Aplicación móvil diseñada para facilitar el descubrimiento de museos y eventos culturales, enfocada en mejorar la experiencia de usuarios interesados en actividades culturales.

El problema

Muchas personas interesadas en actividades culturales no saben qué eventos hay disponibles, dónde encontrarlos o cómo planear una visita de forma sencilla. La información suele estar dispersa en distintas plataformas.

Objetivo

Diseñar una experiencia clara, intuitiva y visualmente atractiva que permita a los usuarios descubrir museos y eventos, conocer detalles importantes y planear su visita fácilmente.

Figma

Proceso UX

Se inició con una investigación básica para entender cómo las personas buscan eventos culturales y qué información consideran más importante.

A partir de esto se crearon wireframes enfocados en simplicidad, jerarquía visual clara y navegación intuitiva.

Diseño UI

El diseño visual busca transmitir una experiencia cultural elegante y accesible, utilizando colores neutros, tipografías limpias y una interfaz amigable.

Resultado final

El resultado es un prototipo funcional que permite explorar museos y eventos, consultar detalles clave y planear visitas de forma sencilla.

Aprendizajes
